#VarATTACHURL

---+++ ATTACHURL -- full URL for attachments in the current topic
   * Syntax: =%<nop>ATTACHURL%=
   * Expands to: =%ATTACHURL%=
   * Example: If you attach a file you can refer to it as =%<nop>ATTACHURL%/image.gif=
